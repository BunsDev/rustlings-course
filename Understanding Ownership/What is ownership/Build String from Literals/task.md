## Build String from Literals

Define the `message` variable of type `String` and fill it with parts 
`part1`, `part2`, and `part3`.

<div class="hint">

The `message` variable must be mutable.
</div>

<div class="hint">

Use `String::from` to initialize the message.
</div>

<div class="hint">

Use `String::push_str` to add new parts to the message.
</div>
